## Aikido Dojo – Web Development Study Project ENG.

This project was developed in 2024 as part of my Web Development studies during the IFRS course. The main goal was to practice building web pages using fundamental front-end technologies, applying concepts of structure, styling, and content organization.

This is an educational project with no commercial purpose, created to consolidate learning and experiment with building a simple institutional website for an Aikido dojo.

## Project Objective

The project focused on:

- Practicing HTML and CSS  
- Structuring a simple institutional website  
- Simulating a real martial arts dojo website  
- Improving layout and content organization  
- Applying basic front-end best practices  

## Technologies Used

-HTML5
-CSS3
